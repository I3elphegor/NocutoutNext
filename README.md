# No cutout Next

This module removes the cutout/notch and with it that extra black bar on top on your phone's display.

The only downside is that if your camera is in the upper-left corner, the clock on the status bar will now be positioned behind it. there are some workarounds for this, but they differs. Tick-tock on the clock... There are a few ways to fix this and move the clock a little to the right, but there are several options, and they vary depending on the manufacturer and Android version, so trying to fix it could result in crashes, boot loops, etc. None of the methods I tried will work across the board, yet. If this bothers you, you can try [Iconify](https://github.com/Mahmud0808/Iconify/).

## Requirements

Enabled Developer Options
Should support rooted Android (maybe 9) 10-16 with either Magisk, KernelSU (with metamodule) or APatch. Android 8 and below will not work.

Tested only on OnePlus 7 Pro + Magisk and OnePlus 10 Pro + KernelSU.

## Usage

`Settings - Develeoper Options - Cutout Emulation - No cutout Next` (name will vary in other languages)
